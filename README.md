# PaoPao

## install mongodb
## start mongodb

    ./bin/mongod -dbpath /var/lib/mongodb &
  
## install dependency

    npm install --save
  
## start

    npm start
or in development model (without restart after modify some code)

    ./bin/supervisor_start.sh
  
## start webpack (build reactjs if you changed the code under reactjs/src)

    webpack
